Files Needed to Run This Project

You need these files:

app.py - The main Python file 

requirements.txt - A file listing all dependencies


Installation & Setup

For the first step, open Command Prompt (write cmd after pressing 'Windows/command + R' buttons together), then write 'cd (your project folder directory)'

For the second step, do this:

For Windows:

Create and activate virtual environment:

python -m venv bug_env
bug_env\Scripts\activate

pip install -r requirements.txt

python app.py



For Mac:

Open Terminal

Create and activate virtual environment:

python3 -m venv bug_env
source bug_env/bin/activate

pip install -r requirements.txt

python3 app.py
